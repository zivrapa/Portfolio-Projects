# UK car accidents 2005-2015 analysis

## General Info

In this project I anallyzed car accidents data from the UK between the years 2005-2015.
The database is not including data about car accidents from Northen-Ireland.

## Technologies

All the project was made by Python.

The libraries is:
* GeoPandas
* Pandas
* NumPy
* Folium
